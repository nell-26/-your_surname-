Система контролю версій Git

Git – розподілена система керування версіями
файлів та спільної роботи.

Проект створив Лінус Торвальдс у 2005 році
для управління розробкою ядра Linux.

Git є однією з найефективніших, надійних і
високопродуктивних систем керування
версіями, що надає гнучкі засоби нелінійної
розробки, що базуються на відгалуженні і
злитті гілок.

Git є вільною системою і випущений під
ліцензією GNU GPL версії 2.
